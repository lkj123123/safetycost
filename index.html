코웨이 산업안전보건관리비

안전한 현장을 위한 스마트 관리 시스템

📋 도움말

🏠 홈 화면 — 프로젝트 관리

  • + 새 프로젝트 : 새로운 공사 프로젝트를 등록합니다.
  • 📋 복사 : 체크박스로 선택한 프로젝트를 그대로 복제합니다. 모든 입력 데이터가 함께 복사됩니다.
  • 선택 삭제 : 체크박스로 선택한 프로젝트를 삭제합니다.
  • 수정 : 해당 프로젝트를 불러와 내용을 수정합니다.
  • 📤 프로젝트 내보내기 : 선택한 프로젝트를 .scpdata 파일로 저장합니다. 다른 PC로 전달할 때 사용합니다.
  • 📥 프로젝트 가져오기 : 저장된 .scpdata 파일을 불러와 목록에 추가합니다.

① 프로젝트 정보 입력

  • 공사업체명, 공사명, 공사사업장명, 공사소재지, 공사대표자, 공사기간을 입력합니다.
  • 모든 정보는 이후 계획서·내역서 문서에 자동으로 반영됩니다.

② 산업안전보건관리비 계상

  • 공사구분 : 건축 / 토목(생산설비) / 중건설 / 특수건설 중 해당 공사를 선택합니다.
  • 공사내역구분 : 견적서에서 재료비·노무비가 구분되어 있으면 "구분됨"을 선택합니다.
  • 총공사금액(VAT 제외) 입력 시 VAT 포함 금액이 자동 계산됩니다.
  • 대상액과 산업안전보건관리비는 입력값이 바뀔 때마다 실시간 자동 계산됩니다.
  • 총공사금액(VAT 포함) 2천만원 미만은 "계상 대상 아님"으로 표시됩니다.

③ 계획 수립 (사용계획서 작성)

  • + 항목 추가로 행을 추가한 뒤, 항목·세부항목·구매일자·단위·수량·단가를 입력합니다.
  • 세부항목은 필수 입력입니다. 세부항목이 없으면 수량·단가를 입력해도 금액이 계산되지 않습니다.
  • 행마다 체크박스를 체크하면 선택 삭제로 여러 항목을 한 번에 삭제할 수 있습니다.
  • 화면 하단에 계상된 산업안전보건관리비 대비 계획금액 비율(%)이 실시간으로 표시됩니다.

④ 사용계획서 확인 & PDF 출력

  • 작성한 계획 내역을 A4 형식의 미리보기로 확인합니다.
  • 작성자 직책·성명을 입력하고 서명란에 마우스(또는 터치)로 서명합니다.
  • 서명 정보는 자동 저장되어 페이지를 이동해도 유지됩니다.
  • 사용계획서 PDF 출력 버튼으로 A4 PDF 파일을 저장합니다.

⑤ 최종 사용내역 정리

  • 실제 구매·사용한 내역을 항목별로 입력합니다. 세부항목은 필수입니다.
  • 📎 증빙자료 : 항목별로 사진을 첨부합니다. 드래그앤드롭 또는 클릭으로 선택하며 이미지 파일만 가능합니다 (파일당 최대 3MB).
  • 계획 수립 내역 가져오기 : 계획 단계에서 작성한 내역을 그대로 불러와 사용내역 작성 시간을 줄일 수 있습니다.
  • 계상된 산업안전보건관리비 대비 사용금액 비율(%)이 실시간으로 표시됩니다.

⑥ 사용내역서 확인 & PDF 출력

  • 작성한 사용내역을 A4 형식의 미리보기로 확인합니다.
  • 작성자 서명 후 사용내역서 최종 PDF 출력 버튼으로 저장합니다.
  • PDF에는 사용내역서 + 항목별 내역서 + 증빙내역(사진)이 모두 포함됩니다.
  • 증빙사진이 없는 항목은 "증빙 없음"으로 표시됩니다.

💡 공통 안내

  • 모든 데이터는 브라우저의 로컬 저장소(localStorage)에 자동 저장됩니다.
  • 같은 브라우저·기기에서는 새로고침 후에도 데이터가 유지됩니다.
  • 다른 기기로 이동하려면 프로젝트 내보내기/가져오기를 사용하세요.
  • 삭제·덮어쓰기 등의 중요 작업은 항상 확인 팝업이 표시됩니다.

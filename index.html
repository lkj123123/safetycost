📋 도움말

🔐 시작 화면 — 저장 방식 선택

  • Google 계정으로 로그인 : Google 계정에 데이터가 저장되어 어떤 기기·브라우저에서도 동일한 데이터를 불러올 수 있습니다.
  • 로컬 저장소 사용 : 로그인 없이 이 기기·브라우저에만 저장됩니다. 다른 기기에서는 데이터가 보이지 않습니다.
  • 선택한 방식은 다음 방문 시에도 자동으로 유지됩니다.
  • 로컬 모드에서 헤더의 클라우드 모드 전환 버튼을 누르면 로컬 데이터를 클라우드로 이전한 뒤 전환할 수 있습니다.

🏠 홈 화면 — 프로젝트 관리

  • + 새 프로젝트 : 새로운 공사 프로젝트를 등록합니다.
  • 📋 선택 복사 : 체크박스로 선택한 프로젝트를 그대로 복제합니다. 모든 입력 데이터가 함께 복사됩니다.
  • 선택 삭제 : 체크박스로 선택한 프로젝트를 삭제합니다.
  • 수정 : 해당 프로젝트를 불러와 내용을 수정합니다.
  • 📤 프로젝트 내보내기 : 선택한 프로젝트를 .scpdata 파일로 저장합니다. 다른 사람에게 전달하거나 백업할 때 사용합니다.
  • 📥 프로젝트 가져오기 : 저장된 .scpdata 파일을 불러와 목록에 추가합니다.
  • 검색창 : 공사명 또는 공사사업장명으로 실시간 검색할 수 있습니다.
  • ☁️ 저장 상태 : 클라우드 모드에서는 검색창 우측에 마지막 저장 시각이 표시됩니다.

① 프로젝트 정보 입력

  • 공사업체명, 공사명, 공사사업장명, 공사소재지, 공사대표자, 공사기간을 입력합니다.
  • 모든 정보는 이후 계획서·내역서 문서에 자동으로 반영됩니다.

② 산업안전보건관리비 계상

  • 공사구분 : 건축 / 토목(생산설비) / 중건설 / 특수건설 중 해당 공사를 선택합니다.
  • 공사내역구분 : 견적서에서 재료비·노무비가 구분되어 있으면 "구분됨"을 선택합니다.
  • 총공사금액(VAT 및 안전관리비 제외) 입력 시 VAT 포함 금액이 자동 계산됩니다.
  • 대상액과 산업안전보건관리비는 입력값이 바뀔 때마다 실시간 자동 계산됩니다.
  • 총공사금액(VAT 포함) 2천만원 미만은 "계상 대상 아님"으로 표시됩니다.

③ 계획 수립 (사용계획서 작성)

  • + 항목 추가로 행을 추가한 뒤, 항목·세부항목·구매일자·단위·수량·단가를 입력합니다.
  • 세부항목은 필수 입력입니다. 세부항목이 없으면 수량·단가를 입력해도 금액이 계산되지 않습니다.
  • 행마다 체크박스를 체크하면 선택 삭제로 여러 항목을 한 번에 삭제할 수 있습니다.
  • 화면 하단에 계상된 산업안전보건관리비 대비 계획금액 비율(%)이 실시간으로 표시됩니다.

④ 사용계획서 확인 & PDF 출력

  • 작성한 계획 내역을 A4 형식의 미리보기로 확인합니다. 화면이 좁으면 자동으로 비율 축소되어 표시됩니다.
  • 작성자 직책·성명을 입력하고 서명란에 마우스(또는 터치)로 서명합니다.
  • 서명 정보는 자동 저장되어 페이지를 이동해도 유지됩니다.
  • 사용계획서 PDF 출력 버튼으로 A4 PDF 파일을 저장합니다.
  • 저장된 PDF는 당사 공사 담당자에게 반드시 제출해야 합니다. 공사 착공 전 또는 담당자가 요청하는 시점에 맞춰 제출하세요.

⑤ 최종 사용내역 정리

  • 실제 구매·사용한 내역을 항목별로 입력합니다. 세부항목은 필수입니다.
  • 📎 증빙자료 : 항목별로 사진을 첨부합니다. 드래그앤드롭 또는 클릭으로 선택하며 이미지 파일만 가능합니다. 첨부된 사진은 자동으로 압축되어 저장됩니다.
  • 계획 수립 내역 가져오기 : 계획 단계에서 작성한 내역을 그대로 불러와 사용내역 작성 시간을 줄일 수 있습니다.
  • 계상된 산업안전보건관리비 대비 사용금액 비율(%)이 실시간으로 표시됩니다.

⑥ 사용내역서 확인 & PDF 출력

  • 작성한 사용내역을 A4 형식의 미리보기로 확인합니다. 화면이 좁으면 자동으로 비율 축소되어 표시됩니다.
  • 작성자 서명 후 사용내역서 최종 PDF 출력 버튼으로 저장합니다.
  • PDF에는 사용내역서 + 항목별 내역서 + 증빙내역(사진)이 모두 포함됩니다.
  • 증빙사진이 없는 항목은 "증빙 없음"으로 표시됩니다.
  • 저장된 PDF는 공사 완료 후 당사 공사 담당자에게 반드시 제출해야 합니다. 증빙자료가 누락되지 않았는지 제출 전 반드시 확인하세요.

💡 공통 안내

  • 데이터는 선택한 저장 방식(클라우드 또는 로컬)에 따라 자동 저장됩니다. 별도의 저장 버튼은 없습니다.
  • 삭제·덮어쓰기 등의 중요 작업은 항상 확인 팝업이 표시됩니다.
  • 인터넷이 끊기면 화면 상단에 오프라인 안내 배너가 표시됩니다. 오프라인 상태에서는 클라우드 저장이 되지 않습니다.
  • 우측 상단 📱 모바일 버튼으로 모바일 최적화 화면으로 전환할 수 있습니다.
